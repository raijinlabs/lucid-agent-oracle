# Plan 3C: Oracle TypeScript SDK — Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish `@lucid-fdn/oracle` v0.1.0 — an overlay-driven Speakeasy-generated TypeScript SDK covering all 15 public Oracle API endpoints with clean resource-oriented naming.

**Architecture:** Separate public repo (`lucid-fdn/oracle-sdk-node`). Speakeasy overlay maps every endpoint to `resource.verb()` names. `speakeasy run` generates 100% of the SDK code from `openapi.json` + `overlay.yaml`. No hand-written wrapper layer.

**Tech Stack:** Speakeasy CLI (authenticated, workspace `raijin-labs-uuf/raijinlabs`), TypeScript, npm (`@lucid-fdn` org)

**Spec:** `docs/superpowers/specs/2026-03-15-plan3c-oracle-sdk-design.md`

### Task 3: Write Speakeasy Overlay

**Files:**
- Create: `oracle-sdk-node/openapi/overlay.yaml`

**Context:** The overlay maps every endpoint to a clean `resource.verb()` SDK method name using `x-speakeasy-group` (resource namespace) and `x-speakeasy-name-override` (method name). It also overrides the server URL to `https://api.lucid.foundation` and hides the `/health` endpoint with `x-speakeasy-ignore`. The overlay targets use JSONPath syntax per the Overlay Specification 1.0.0.

All 15 endpoint mappings:
| Endpoint | Group | Method |
|----------|-------|--------|
| `GET /v1/oracle/feeds` | feeds | list |
| `GET /v1/oracle/feeds/{id}` | feeds | get |
| `GET /v1/oracle/feeds/{id}/methodology` | feeds | methodology |
| `GET /v1/oracle/feeds/{id}/history` | feeds | history |
| `GET /v1/oracle/agents/search` | agents | search |
| `GET /v1/oracle/agents/leaderboard` | agents | leaderboard |
| `GET /v1/oracle/agents/model-usage` | agents | modelUsage |
| `GET /v1/oracle/agents/{id}` | agents | get |
| `GET /v1/oracle/agents/{id}/metrics` | agents | metrics |
| `GET /v1/oracle/agents/{id}/activity` | agents | activity |
| `GET /v1/oracle/protocols` | protocols | list |
| `GET /v1/oracle/protocols/{id}` | protocols | get |
| `GET /v1/oracle/protocols/{id}/metrics` | protocols | metrics |
| `GET /v1/oracle/reports/latest` | reports | latest |
| `POST /v1/oracle/reports/verify` | reports | verify |

```typescript
import { describe, it, expect } from 'vitest'
import { LucidOracle } from '../src/index.js'

// Anonymous client (no API key) — free-tier endpoints should work
const oracle = new LucidOracle({
  serverURL: process.env.SERVER_URL ?? 'http://localhost:4040',
})

// Authenticated client (if LUCID_ORACLE_API_KEY is set)
const authedOracle = process.env.LUCID_ORACLE_API_KEY
  ? new LucidOracle({
      apiKey: process.env.LUCID_ORACLE_API_KEY,
      serverURL: process.env.SERVER_URL ?? 'http://localhost:4040',
    })
  : null

describe('feeds', () => {
  it('list() returns feed array', async () => {
    const result = await oracle.feeds.list()
    expect(result).toBeDefined()
    // Response should contain data array
    expect(Array.isArray(result.data ?? result)).toBe(true)
  })

  it('get() returns a single feed', async () => {
    const feeds = await oracle.feeds.list()
    const feedList = feeds.data ?? feeds
    if (Array.isArray(feedList) && feedList.length > 0) {
      const feed = await oracle.feeds.get({ id: feedList[0].id })
      expect(feed).toBeDefined()
    }
  })

  it('methodology() returns methodology text', async () => {
    const feeds = await oracle.feeds.list()
    const feedList = feeds.data ?? feeds
    if (Array.isArray(feedList) && feedList.length > 0) {
      const result = await oracle.feeds.methodology({ id: feedList[0].id })
      expect(result).toBeDefined()
    }
  })
})

describe('agents', () => {
  it('search() returns paginated response', async () => {
    const result = await oracle.agents.search({ q: 'test' })
    expect(result).toBeDefined()
    expect(result.data).toBeDefined()
    expect(result.pagination).toBeDefined()
  })

  it('leaderboard() returns paginated response', async () => {
    const result = await oracle.agents.leaderboard({})
    expect(result).toBeDefined()
    expect(result.data).toBeDefined()
  })
})

describe('protocols', () => {
  it('list() returns protocol array', async () => {
    const result = await oracle.protocols.list()
    expect(result).toBeDefined()
    expect(Array.isArray(result.data ?? result)).toBe(true)
  })
})

describe('reports', () => {
  it('latest() returns report or empty', async () => {
    const result = await oracle.reports.latest()
    expect(result).toBeDefined()
  })
})

describe('auth behavior', () => {
  it('anonymous client can call free-tier endpoints', async () => {
    // Should not throw
    const result = await oracle.feeds.list()
    expect(result).toBeDefined()
  })

  it('anonymous client gets 403 on pro-tier endpoint', async () => {
    try {
      // agents.metrics requires pro tier
      await oracle.agents.metrics({ id: 'any-agent-id' })
      // If no agents exist, this may 404 instead of 403 — both are acceptable
    } catch (err: any) {
      expect([403, 404]).toContain(err.statusCode ?? err.status ?? err.httpMeta?.response?.status)
    }
  })

  it('authenticated client can call pro-tier endpoint', async () => {
    if (!authedOracle) {
      console.log('Skipping: LUCID_ORACLE_API_KEY not set')
      return
    }
    // Need a real agent ID — get one from leaderboard
    const lb = await authedOracle.agents.leaderboard({ limit: 1 })
    const agents = lb.data ?? []
    if (agents.length === 0) {
      console.log('Skipping: no agents in leaderboard')
      return
    }
    const result = await authedOracle.agents.metrics({ id: agents[0].id })
    expect(result).toBeDefined()
  })
})

describe('pagination', () => {
  it('leaderboard pagination round-trip', async () => {
    const page1 = await oracle.agents.leaderboard({ limit: 1 })
    expect(page1.data).toBeDefined()

    // Field may be next_cursor or nextCursor
    const cursor = page1.pagination?.next_cursor ?? page1.pagination?.nextCursor
    if (cursor) {
      const page2 = await oracle.agents.leaderboard({ limit: 1, cursor })
      expect(page2.data).toBeDefined()
    }
  })
})

describe('error contract', () => {
  it('404 on nonexistent agent', async () => {
    try {
      await oracle.agents.get({ id: 'nonexistent-agent-id-12345' })
      expect.fail('Expected error for nonexistent agent')
    } catch (err: any) {
      // SDK error should expose status code
      const status = err.statusCode ?? err.status ?? err.httpMeta?.response?.status
      expect(status).toBe(404)
    }
  })

  it('400 on search with missing required param', async () => {
    try {
      // q is required for agents.search — omitting it should return 400
      await oracle.agents.search({} as any)
      expect.fail('Expected validation error for missing q param')
    } catch (err: any) {
      const status = err.statusCode ?? err.status ?? err.httpMeta?.response?.status
      expect(status).toBe(400)
    }
  })
})
```

````markdown
## Pagination

Cursor-paginated endpoints return `{ data, pagination }`:

```typescript
let cursor: string | undefined
do {
  const page = await oracle.agents.search({ q: 'lucid', cursor })
  for (const agent of page.data) {
    console.log(agent)
  }
  cursor = page.pagination.next_cursor ?? undefined
} while (cursor)
```
````
